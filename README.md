# Restaurant Insights Analysis

This project analyzes a global restaurant dataset to uncover trends in pricing, ratings, cuisines, and geographic distribution. It includes exploratory data analysis, descriptive statistics, feature engineering, and geospatial visualization. The goal is to convert raw data into meaningful insights that reflect customer behavior, restaurant patterns, and city-level trends.
